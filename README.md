# 🌍 Urbanization and Deforestation Analyzer

An AI-powered geospatial analysis application that compares **urban expansion** and **deforestation** over different years using **Google Earth Engine**, **satellite imagery**, and an interactive web dashboard. The project helps visualize land-cover changes and supports environmental monitoring and sustainable urban planning.

---

## 📌 Features

- Analyze urbanization and deforestation over time.
- Compare satellite imagery from different years.
- Interactive map visualization.
- Detect changes in urban and forest land cover.
- User-friendly web interface.
- Powered by Google Earth Engine for large-scale geospatial analysis.

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/baderamesh2005/Urbanization-and-Deforestation-Analyzer.git
```

### Navigate to the project

```bash
cd Urbanization-and-Deforestation-Analyzer
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install required packages

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

## 🌎 How It Works

1. Select the area of interest.
2. Choose the years to compare.
3. Google Earth Engine retrieves satellite imagery.
4. The application analyzes land-cover changes.
5. Urban and forest changes are displayed on an interactive map with statistics.
